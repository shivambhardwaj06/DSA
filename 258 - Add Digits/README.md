# 258. Add Digits

## Problem
Given an integer num, repeatedly add all its digits until the result has only one digit.

### Approach 1
- Repeatedly calculate digit sum.
- Time: O(log n × iterations)
- Space: O(1)

### Approach 2
- Uses Digital Root formula.
- Time: O(1)
- Space: O(1)

## Concepts
- Modulo (%)
- Integer Division (/)
- Digital Root
